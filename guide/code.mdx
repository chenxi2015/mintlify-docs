---
title: '错误码说明'
description: '任何与开放平台的接口之间调用都使用错误码作为接口异常反馈依据, 以下是所有状态码, 请开发人员留意。'
icon: 'map'
---

| 响应码（errcode） | 响应码描述（message） | 处理建议 |
| :--- | ---- | :---- |
| 0  | 	成功  |	服务正常 |
| 10001  | 	请求被禁止  |	查看请求参数是否按照文档中的名称、类型、是否必传传输 |
| 10002  | 	用户未认证  |	查看请求参数是否按照文档中的名称、类型、是否必传传输 |
| 10006  | 	参数验证失败  |	查看请求参数是否按照文档中的名称、类型、是否必传传输 |
