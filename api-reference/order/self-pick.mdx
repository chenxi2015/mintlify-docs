#### 请求方式：`POST`
#### 请求类型：`application/json`，`application/x-www-form-urlencoded`

#### 请求地址：
```
/v1/order/self-pick
```

    <ResponseField name="timestamp" type="integer">
     当前时间戳（秒）参数示例：1741478746
